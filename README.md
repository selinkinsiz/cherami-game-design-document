# Game Design Document (GDD)

## Project Overview
This repository contains the portfolio version of a comprehensive Game Design Document outlining the concept, mechanics, systems, and structural framework of the project.

The document defines gameplay structure, progression systems, world design, user experience elements, and commercialization considerations.
